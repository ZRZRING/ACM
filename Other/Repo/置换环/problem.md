### 题目描述

给定长度为 $n$ 的排列 $a_i$，你每次可以选择任意两个数交换，问最少交换多少次可以使得排列升序。

### 输入描述

第一行一个整数 $T$ $(T\leq 5)$，表示数据组数。

对于每组数据，第一行一个整数 $n$ $(1\leq n\leq 100)$，第二行 $n$ 个整数 $a_i$ $(1\leq a_i\leq n, \forall i \not = j, a_i \not = a_j)$，表示一个排列。

### 输出描述

一行一个整数 $ans$ 表示答案

### 样例 1

```
3
3 1 2
```

```
2
```
